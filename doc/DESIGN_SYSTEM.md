# Design System & Theme

## Visual Identity: "Midnight Serenity"
Derived from the application logo, the design language dictates a calm, spiritual, and premium atmosphere.

## Color Palette

### Primary Colors
- **Midnight Blue (`#020617`)**:
  - *Usage*: Main Backgrounds, Header, Footer.
  - *Vibe*: Deep, calming, battery-efficient for OLED.
  
- **Soft Teal (`#2dd4bf`)**:
  - *Usage*: Primary Buttons, Active States, Progress Bars.
  - *Vibe*: Fresh, modern, friendly contrast to the dark background.

- **Golden Hour (`#fbbf24`)**:
  - *Usage*: "Next Prayer" Highlight, Critical Alerts, Icons.
  - *Vibe*: Warmth, urgency without aggression, spiritual connection.

### Neutral Colors
- **Surface (`#1e293b`)**: Cards background (slightly lighter than main bg).
- **Text Primary (`#f8fafc`)**: High readability text (Off-white).
- **Text Secondary (`#94a3b8`)**: Subtitles, inactive elements.

## Typography
- **Font Family**: `Inter` (English) + `Noto Kufi Arabic` (Arabic).
- **Weights**:
  - Regular (400) for body text.
  - Medium (500) for prayer names.
  - Bold (700) for "Next Prayer" countdown.

## UI Components
- **Cards**: Rounded corners (`1rem`), subtle border (`1px solid #334155`), backdrop blur if needed.
- **Buttons**: Pill-shaped, flat design, slight hover elevation.
- **Icons**: `Lucide React` with thin strokes (1.5px) to match the elegant aesthetic.
