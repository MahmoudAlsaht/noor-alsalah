# Noor Al-Salah (نور الصلاة)

تطبيق مواقيت صلاة عائلي بسيط وجميل، مصمم لتلبية احتياجات المستخدمين في الأردن (إربد) مع دعم التنبيهات والويدجت والتواريخ الهجرية والميلادية.

## المميزات الرئيسية
*   **مواقيت دقيقة:** حساب أوقات الصلاة بناءً على الطريقة الأردنية (وزارة الأوقاف).
*   **تصميم جديد ومحسن:** واجهة مستخدم مبسطة وجذابة تدعم الوضع الداكن.
*   **التواريخ:** عرض التاريخ الهجري والميلادي بشكل أنيق في التطبيق والويدجت (تقويم أم القرى).
*   **تنبيهات مخصصة:** تحكم كامل في تنبيهات كل صلاة (وقت الدخول، قبل انتهاء الوقت) مع أصوات متعددة.
*   **منبه ذكي:** شاشة تنبيه مخصصة (Full Screen Alarm) عند يحين وقت الصلاة، مع إيقاف تلقائي.
*   **الويدجت (Widget):** ويدجت ذكي يعرض الصلاة القادمة، العد التنازلي، والتواريخ الهجرية والميلادية بتصميم عصري.
*   **نسخة الويب:** نسخة ويب مبسطة وخفيفة لعرض المواقيت وتحميل التطبيق فقط.

## التغييرات في الإصدار v0.1.23
*   ✅ **أيقونة جديدة:** تصميم مئذنة بسيط وأنيق بخلفية داكنة.
*   ✅ **تحديث الويدجت:** عرض التواريخ الهجرية والميلادية في المنتصف وفواصل بصرية أنيقة.
*   ✅ **إصلاح الصوت:** حل مشكلة تداخل الأصوات عند تغيير نغمة التنبيه في الإعدادات.
*   ✅ **تحسين الويب:** إزالة طلبات الأذونات وشاشات الإعداد من نسخة الويب لتكون أسرع وأبسط.
*   ✅ **إخفاء أدوات المطور:** أدوات الاختبار تظهر الآن فقط في بيئة التطوير.

## المشاكل المعروفة (Known Issues)

## التطوير (Development)

### المتطلبات
*   Node.js 20+
*   Android Studio (لبناء نسخة الأندرويد)
*   Capacitor CLI

### أوامر مفيدة
```bash
# تشغيل نسخة الويب (التطوير)
npm run dev

# بناء نسخة الإنتاج
npm run build

# مزامنة مع الأندرويد
npx cap sync android

# فحص الأكواد
npm run lint
```

## التقنيات المستخدمة
*   Next.js 16 (App Router)
*   Capacitor (Native runtime)
*   Lucide React (Icons)
